## Setup for Server/Client
```shell
cmake -S . -B build/ &&
cmake --build build/ &&
cmake --install build
```

### Run Server
```shell
cd server/extra/
uvicorn server:app --reload
```


### Run Client
`python3 client/src/gui.py`
